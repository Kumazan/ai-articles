# Session Walkthrough Report Template

## 目標
- 使用者想達成什麼

## 起始狀態
- 已知環境
- 症狀 / 錯誤

## Walkthrough 時間線
1. 第一階段：確認現況
2. 第二階段：排除錯誤假設
3. 第三階段：找到真正根因
4. 第四階段：實作修正
5. 第五階段：驗證成功

## 關鍵判斷
- 哪個訊號最重要
- 哪個假設後來被推翻
- 哪個設定是最後的關鍵

## 實際修改 / 指令
- 設定檔修改
- 關鍵指令
- 服務變更

## 最終結果
- 成功狀態
- 可重複使用入口

## 後續 SOP
- 新裝置怎麼處理
- 下次遇到類似問題怎麼排查
