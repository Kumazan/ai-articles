# Bakusai Reader — 輸出格式與寫作原則

## 風向總結格式（推薦）

當使用者想知道「這家最近在聊什麼」、「近期 vibe」、「值不值得去」時，使用此格式。

### 建議格式

```md
### 店名

**最近 N 則的感覺：**
- ...
- ...
- ...

**整體印象：**
- ...
- ...
- ...

**一句話總結：**
- ...
```

---

## 寫作原則

- 重點是整理**近期風向 / 社群氣質 / 場內 vibe**，不是逐字翻譯。
- 先看最近 50–100 則的主題分布：
  - 即時約人很多？
  - 固定人物 drama 很多？
  - 規矩 / 巡場 / 店員抱怨很多？
  - 外型門檻 / 客群條件焦慮很多？
- 允許有觀察感的中文，但**不捏造未出現的資訊**。
- 樣本偏少（3–10 則）時，明確寫「樣本偏少」。
- 串很活但很雜 → 「高流量聊天室型」
- 內容多為人物八卦 → 「人物誌 / 小圈圈 drama 感很重」
- 內容多為即時報位、身高體重年齡、房號 → 「即時約砲板 / call 台」

---

## 逐則翻譯格式

當使用者需要原文翻譯時：

```
#XXX [日期時間]
（翻譯）
```

- 保留 `#編號` 與 `日期時間`
- 場所黑話要意譯（見 terms.md）
