# Itinerary Structure Template

Use this template for each day in `trips/{trip-name}/itinerary.md`.

```markdown
# Day X — {Date} — {Location}

## Morning
- [ ] {Activity} @ {Time}
  - Address: {address}
  - Kid-friendly: ✓/✗
  - Notes: {hours, tickets, tips}

## Afternoon
- [ ] {Activity} @ {Time}
  - Address: {address}
  - Notes: {reservation, cost}

## Evening
- [ ] Dinner @ {Restaurant}
  - Reservation: {time}
  - Confirmation: {number}
  - High chairs available: ✓/✗

## Logistics
- Transport: {how to get there — metro line, bus number, taxi estimate}
- Accommodation: {check-in time if applicable}
- Backup plan: {if weather/energy fails}

## Notes
- Must-dos: {list}
- Nice-to-haves: {list}
- Emergency contacts: {local hospital, embassy, hotel front desk}
```

## Usage Rules

- Keep 2–3 hours buffer daily — don't fill every slot.
- Mark activities as **must-do** vs **nice-to-have** before the day starts.
- Max 3 planned activities per day to leave room for discovery.
- Afternoon nap slot for family/group trips with young children.
- Log actual times/costs inline after completing each activity (for future reference).
