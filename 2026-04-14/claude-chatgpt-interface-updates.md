## 摘要

- Claude Code 傳出要做更完整的桌面工作台，介面重點放在 Plan、Tasks、Diff 與多倉庫協作
- OpenAI 也在把 ChatGPT、Atlas、Codex 往同一個桌面 superapp 收攏，並用 Scratchpad 分派平行任務
- 這波競爭重點已經從「哪個模型最強」轉到「誰的工作流最順、最像真正的作業系統」
- Anthropic、OpenAI、Meta、Perplexity 都在把 AI 往更大權限、更深整合的方向推
- 安全、可驗證性與權限邊界，接下來只會變成更大的產品問題
- 對開發者來說，介面設計和代理編排正在變成跟模型選型一樣重要的決策

## 桌面戰場，正在重做

這一期的核心很簡單，AI 產品戰爭已經不只是模型之爭，而是桌面介面之爭。Anthropic 和 OpenAI 都在把各自的工具往更完整的工作台方向推，目標不是多一個聊天框，而是一個能接管規劃、瀏覽、寫作、除錯與多步驟任務的主控台。

Claude Code 傳出的「Epitaxy」更新，像是把 Cowork 式的桌面布局直接拉進開發者工作流。Plan、Tasks、Diff 幾個面板一起上桌，再加上多倉庫支援與 Coordinator Mode，意思很明確，未來不是叫模型幫忙回答，而是讓它在多個子任務之間自己編排。

OpenAI 這邊的方向也差不多。TestingCatalog 的報導指出，ChatGPT、Atlas 和 Codex 正在合併成一個更像桌面 superapp 的體驗，Scratchpad 介面可以把 TODO 拆成平行的 Codex 任務。這不是單純加功能，而是把聊天、瀏覽器與程式工作流綁成同一個產品邏輯。

## 其他更新

- Anthropic 也被傳出在測試 Claude Tasks Mode 的 Build 選項，介面從空白畫布改成 Research、Analyze、Build、Write、Plan 五個工作流入口，明顯對標 Replit 和 v0
- Sam Altman 的住處在 48 小時內接連遭到汽油彈與槍擊，雖然和產品本身無關，但也把 AI 巨頭周邊的社會壓力與對立拉到檯面上
- Anthropic 同時把 Mythos Preview 收進 Glasswing 資安聯盟，不公開釋出，改讓 AWS、Apple、Cisco、CrowdStrike、Linux Foundation 與其他夥伴拿去做防禦性漏洞研究
- Meta 的 Muse Spark 則把閉源模型塞進 Meta AI 的整體產品線，還加入 Contemplating mode，讓多代理並行推理直接變成產品能力
- Anthropic 的營收也繼續上衝，年化 ARR 據報已到 300 億美元，1,000 家以上大企業客戶的成長速度仍然很快
- OpenAI 替 Codex 開出 100 美元的 Pro 方案，夾在 20 美元 Plus 與 200 美元 Pro 之間，讓重度使用者有新選項
- Apple 傳出正在測試四種智慧眼鏡框型，預計 2027 年推出，首代先不做顯示器，重點在外觀與感測整合
- Perplexity Computer 也在往個人 CFO 的方向擴張，接上 Plaid 後，已可讀取支票、儲蓄、信用卡與貸款資料
- xAI 則傳出財務主管 Anthony Armstrong 離職，顯示馬斯克系 AI 團隊的人事震盪還在繼續
- 美國法院也維持 Anthropic 被排除在國防合約外的現況，讓資安與軍工的界線繼續卡在法律與政治中間

## 這波更新透露的訊號

第一，產品邏輯正在從對話框轉向工作台。第二，代理不再只是「幫你寫一段」，而是要能接手整段流程。第三，最重要的競爭點開始變成 UI、權限、狀態管理與驗證機制，而不是單一 benchmark。

Anthropic 也沒有只盯著開發者圈。從 Glasswing 資安聯盟到 Mythos Preview 的限制釋出，都顯示它把高能力模型當成需要被嚴格包裝的基礎設施，不是隨便丟到公開市場的玩具。Meta 的 Muse Spark 則走相反路線，直接把閉源模型塞進更大的產品整合裡，連多代理推理也一起做進去。

Perplexity 把 Computer 往個人理財方向推，Apple 在測試智慧眼鏡外型，OpenAI 甚至把 Codex 往 100 美元的 Pro 方案推進去。整體看下來，大家都在搶同一件事，誰能把 AI 變成真正的日常操作層。

## 延伸評論：介面會決定誰留下來

這篇最值得記住的，不是某個功能多酷，而是 AI 產品的勝負正在往「誰比較像系統」移動。當模型能力差距開始縮小，真正拉開差距的就會是工作流、記憶、任務切換、權限與可回溯性。

也就是說，未來的主戰場不會只在模型榜單上，而是在使用者每天打開哪個介面、把多少工作交給哪個代理、以及哪個產品能在不失控的前提下吞下更多權限。這才是介面更新背後真正的野心。
